# SimSort AI Platform

Basic dashboard for SimSort AI platform.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Add your Gemini API key to `.streamlit/secrets.toml`
3. Run the dashboard: `streamlit run app.py`

## Videos

Place pre-recorded videos in the `videos/` folder:
- eval_success.mp4
- eval_failure_recovery.mp4
- training_demo.mp4

## Deployment

Use the included Dockerfile for Coolify deployment.
